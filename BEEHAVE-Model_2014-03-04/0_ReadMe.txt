To run BEEHAVE, please download and install NetLogo (http://ccl.northwestern.edu/netlogo/) first.
Open 1_BEEHAVE-MODEL_Beehave2013.nlogo, click "Default", "Setup" and "Run".
Click "varroa" and "beekeeping" to add varroa mites and beekeeping practices.
Play with the model by changing e.g. nectar quantity (QUANTITY_R/G_l), concentration (CONC_R/G), pollen availability (POLLEN_R/G_kg), and distance of the two default food patches.
The user manual (2_BEEHAVE_Manual.pdf) provides a detailed description of the BEEHAVE model.
